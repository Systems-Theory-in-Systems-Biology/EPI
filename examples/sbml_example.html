SBML Models

The SBMLModel loads a biological model from an external file written in the sbml standard format. It generates the forward and jacobian method automatically and derives the parameter and data dimension from the sbml model.

The two example models are included in eulerpi/examples/sbml/.

Specialities

  • Super simple setup

  • No need to write any model code

Here’s a code snippet to load your own sbml model and to do the parameter inference:

import numpy as np
from eulerpi.core.model import SBMLModel
from eulerpi.core.inference import inference

# This line is needed for multiprocessing in python
if __name__ == "__main__":
    central_param = np.array([1.0, 1.0])  # initial guess, evaluation must have nonzero density
    param_limits = np.array([[0.0, 2.0], [0.0, 2.0]])
    param_ids = ['k1', 'k2']
    timepoints = np.array([1.0])

    model = SBMLModel(sbml_file='model.xml',
                    central_param=central_param,
                    param_limits=param_limits,
                    timepoints=timepoints,
                    param_ids=param_ids)
    inference(model, 'data.csv')

The attribute param_names contains the names of the parameters in the sbml model, for which the inference should be performed. Per default it contains all parameters from the sbml model file.

Note

For the SBML Standard see https://sbml.org/. You can visualize sbml files with https://sbml4humans.de/.
