EPI can be used in several ways and can be applied to many different problems. We will show how EPI can be applied to a simple, one-dimensional problem; a high-dimensional problem; a sbml model; and a problem defined through external C++ code. For more details on the mathematics behind the epi algorithm, we refer to our published paper. For more details on how to use EPI, we refer to our Tutorial page.

General

In all exaples we assume a given (discrete) data probability distribution \(y_i \sim Y\). The goal is to find the parameter probability distribution \(Q\) satisfying \(Y = s(Q)\).

Temperature Model

The temperature model is contained in epic/example_models/temperature. The model \(y_i(q_i)=60 \cos(q_i)-30=s(q_i)\) describes the temperature for a place on the earth \(y_i\) by using the latitude coordinates \(q_i\).
